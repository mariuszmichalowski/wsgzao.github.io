前言

到了新加坡再也不用考虑科学上网的问题，加上 macOS 跨平台浏览器做得比较成熟的自然是 Google Chrome 和 Firefox，在 Windows 平台用了十几年的 Maxthon，期间也尝试过无数第三方浏览器，傲游浏览器努力改变着世界却也渐行渐远，有着同样命运的应该还包括我从 M8 用到 MX2 的魅族吧。回到 Google Chrome 浏览器插件和油猴脚本推荐正题，Google Chrome 除了最重要的快以外，更重要的是 Chrome Web Store 上一系列好用插件，我很喜欢 Chrome 简单纯粹的设计，所以在插件的推荐上分为 通用 和 开发者 两部分来写方便区分，大部分 Chrome 可用的插件 Firefox 也同样适用，如果有遗漏或有更好用的同类型插件和脚本欢迎留言分享。

通用插件

SimpleExtManager -

Violentmonkey - 代替 Tampermonkey 和 GreaseMonkey，样式简洁管理更方便
Tampermonkey - 老牌的油猴管理工具，哪款用得顺手选择哪个

沙拉查词 - 聚合词典划词翻译 - 我个人觉得这是目前 Chrome 划词翻译插件中体验最好的

Grammarly for Chrome - 智能检查英文语法拼写错误，在 Web 写邮件和文档时会发现它的重要价值

Evernote Web Clipper - 中文称为印象笔记，平时会把优质的内容保存到 Evernote 中，虽然没有 OneNote 功能强大但胜在简单方便可跨平台同步

破解右键锁 - 比 Enable Copy 更加强大，不过大众点评的评论好像无解

开发者插件

JSONViewer - 一个页面格式化多条 JSON，提升工作效率
